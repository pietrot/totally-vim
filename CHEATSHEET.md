## FOLDING
#### CONFIGURATION
```
" Folding options
set foldenable
set foldmethod=syntax
set foldlevel=999
set foldcolumn=0
```
#### HOTKEYS
- `zo` opens a fold at the cursor
- `zO` opens all folds at the cursor
- `zc` close one fold under the cursor
- `zC` close all folds under the cursor recursively
- `za` toggle fold

## COPY/PASTE
- `gv` recover previous selection
